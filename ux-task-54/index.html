UX Phase 02: Mapping

54. Customer Journey Map Matrix

User Goal: Finding and buying a specialized software course online.

Stage 1

Discovery

🔍 User Actions

Searches on Google; browses course listings via social links.

❌ Pain Points

Too many options; hard to compare prices at a glance.

💡 Opportunities

Create a clean, 3-tier matrix pricing layout.

Stage 2

Evaluation

🔍 User Actions

Reads curriculum details; watches short teaser previews.

❌ Pain Points

Video players lag; hard to see exact lesson time requirements.

💡 Opportunities

Add precise time badges next to each module header.

Stage 3

Purchase

🔍 User Actions

Taps register; inputs billing fields; submits checkout.

❌ Pain Points

Unexpected transaction fields or hidden fees pop up.

💡 Opportunities

Ensure total cost transparency inside the basket step.
